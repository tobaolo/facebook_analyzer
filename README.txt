This program takes downlaoded facebook messenger conversations and analyzes groupchats. 

To run this program, open the file and one your terminal to the appropriate directory. Run "python facebook-message-analyzer.py"
to run the program. 

The program will propmt you for a name. This is one of the methods of the program that prints all messages sent by that person.
The names are determined by who is in the groupchat. To test this, you can enter any of the following names:
    Toba Olokungemi
    Bola Hambolu
    Ugo Aguwa
    Jess Ekeya
    Angelica Chima
    The messages sent by the name typed will be printed in the terminal

Then the subsequent graphs will appear in a new window. You must close the current graph before the next graph will po up.
